#Quiz Game System

##Project Description
The Quiz Game System is a command-line Python application that allows users to take interactive quizzes on different topics. The system loads questions from a file, presents them to the user one at a time, tracks their answers, calculates their score, and displays their final results. The project demonstrates the use of Python functions, file handling, data structures, and Git collaboration between two partners.

##Features
- Create and manage a question bank with at least 20 quiz questions
- Save questions to a text file
- Load questions from the text file
- Add new questions through terminal input
- Filter questions by difficulty level (easy, medium, hard)
- Load and shuffle quiz questions
- Allow users to select number of questions
- Run the quiz and validate answers
- Track user score during the quiz
- Display final quiz results and percentage
- Assign grades based on performance
- Maintain a leaderboard with top 5 scores

##Step-by-step Instructions To Run The Project Locally
1. Clone the repository
2. Navigate into the project folder
3. Run the program
4. Follow the instructions in the terminal to start the quiz
